# Functional Stability Theory (FST)

A unified framework connecting thermodynamic stability, game theory, and mathematical physics.

## Structure

- **framework/** -- The Renormalized Free-Energy Framework (RFEP, Pattern A)
- **domain-proofs/** -- Domain-specific instantiations (Navier-Stokes, Yang-Mills, Dark Energy, Turbulence, P-vs-NP, BSD, Hodge)
- **applications/** -- Empirical applications (FST-I Particles, FST-II Chemistry, FST-III Biology)
