# Functional Stability Theory (FST)

A unified framework connecting thermodynamic stability, game theory, and mathematical physics.

## Structure

- **domain-proofs/** — Domain-specific instantiations (Dark Energy, Navier-Stokes, Yang-Mills, Turbulence, P-vs-NP, BSD, Hodge)
- **applications/** — Empirical applications (FST-I Particles, FST-II Chemistry, FST-III Biology)
- **fst_references.bib** — Shared bibliography

> **Note:** Only papers published on Zenodo are included in the repository.
> Unpublished domain proofs and applications are listed in `.gitignore` and will be
> added after Zenodo publication.
